## üéØ Common Workflows

### "I Just Got a New Android Phone and Want to Run a Server"
1. Read: [QUICK_START_OPTIMIZE.md](QUICK_START_OPTIMIZE.md) - 5 min
2. Open HoneyBee ‚Üí Optimize tab
3. Let device specs detect
4. Select "High-End Device" (if 6GB+) or "Medium" (if 4GB)
5. Click Apply
6. Restart server from Dashboard
7. Monitor TPS in Metrics tab
8. Done! üéâ

### "My Server is Laggy and I Need to Fix It"
1. Open HoneyBee ‚Üí Optimize tab
2. Check device specs
3. Try "Reset to Default"
4. If still laggy, select lower profile
5. Enable "Reduce View Distance"
6. Click Apply ‚Üí Restart
7. Check Metrics tab
8. If better: Done! If not: Try [PERFORMANCE_OPTIMIZATION.md](PERFORMANCE_OPTIMIZATION.md#troubleshooting)
